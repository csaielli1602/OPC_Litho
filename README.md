# OPC & Resolution Enhancement Explorer

A single HTML file that shows how optical proximity correction (OPC) makes it possible to print chip features smaller than the wavelength of light doing the printing. I built this after touring semiconductor fabs and research facilities and wanting a visual, hands-on way to see why mask shapes get deliberately distorted to correct for the blur of the optics. Open `index.html` in any browser, no install, no build step, and toggle illumination modes, OPC level, and correction features to watch the printed shape sharpen or degrade. It's meant for anyone who wants an intuitive, visual feel for this part of chip manufacturing rather than a production-accurate model. Companion piece to `Silicon_Pull/`; source comments explain what's physically exact versus stylized.
