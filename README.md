# Silicon Czochralski Crystal-Pulling Simulator

This tool visualizes a simplified version of the Czochralski process, the method used to grow the single-crystal silicon boules that every chip wafer gets sliced from. I built this after touring semiconductor fabs and research facilities and wanting a way to actually feel how pull rate, rotation, and heat trade off against each other, instead of just reading the equations. Open `index.html` in any browser, no install, no build step, and adjust the sliders to watch the boule narrow, flare, or drift in real time. It's meant for anyone who wants a more intuitive, visual feel for how this part of chip manufacturing actually works, not a production-accurate model. Full breakdown of which equations are exact versus simplified is commented directly in the source.
